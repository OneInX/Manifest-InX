# Code of Conduct

## Expected behavior

- Be respectful and professional.
- Assume good faith; disagree with ideas, not people.
- Keep discussions on-topic and constructive.
- Respect privacy and security; do not share sensitive information.

## Unacceptable behavior

- Harassment, discrimination, or hate speech.
- Personal attacks, trolling, or sustained disruption.
- Publishing othersâ€™ private information without permission.
- Sexual content or unwanted advances.
- Threats or encouragement of violence.

## Scope

This Code of Conduct applies in all project spaces, including GitHub issues, pull requests, code reviews, and discussions.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, content removal, temporary or permanent restrictions, or banning contributors from project spaces.

## Reporting

Report Code of Conduct concerns privately using the same channel as `SECURITY.md`: GitHub Security Advisories / private vulnerability reporting for this repository.

Include relevant context (links, screenshots, logs) so maintainers can assess and respond.

Maintainers will acknowledge receipt within 7 days.
