# Support

This repository is provided as-is.

## Where to route requests

- Bugs: GitHub Issues (use the Bug Report template)
- Feature requests: GitHub Issues (use the Feature Request template)
- Security reports: GitHub Security Advisories (do not file a public issue)

## Out of scope

- Usage coaching
- Customization requests
- Interpretation of engine outputs

## Response expectations

- No guaranteed response time.
- Issues and requests may be closed without a reply.
- Security reports may receive a private acknowledgement when triage occurs.
